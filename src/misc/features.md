# 功能特性

本工具以 [网站](https://down.mptext.top/) 形式提供，以下是当前支持的功能列表。

## 核心功能

| 功能 | 说明 |
|------|------|
| 搜索公众号 | 支持关键字搜索，添加到管理列表 |
| 同步文章列表 | 批量拉取公众号的全部历史文章链接 |
| 下载文章内容 | 通过代理池并发下载，支持断点续传 |
| 抓取元数据 | 阅读量、点赞数、转发量（需 Credentials） |
| 抓取评论 | 精选评论和回复内容（需 Credentials） |
| 本地缓存 | 已下载数据存储在浏览器 IndexedDB 中，避免重复下载 |

## 导出格式

| 格式 | 说明 |
|------|------|
| HTML | 高保真还原，打包图片和样式到 ZIP |
| Excel | 表格形式，包含元数据 |
| JSON | 结构化数据，适合程序处理 |
| Markdown | 纯文本标记格式 |
| TXT | 纯文本内容 |
| Word (DOCX) | 适合离线阅读和打印 |

## 高级功能

- 文章过滤：按作者、标题、发布时间、是否原创等条件筛选
- 合集下载：支持按公众号合集批量下载文章
- 单篇下载：通过文章 URL 直接下载单篇文章
- API 接口：提供 RESTful API，支持第三方集成
- 图片分享消息下载
- 视频分享消息下载

## 部署方式

- Docker 部署（推荐）
- Cloudflare Pages 部署
- Vercel 一键部署
- 本地开发运行
