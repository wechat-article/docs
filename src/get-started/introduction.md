# 介绍

**公众号文章下载工具** 是一款开源免费的在线工具，支持批量下载微信公众号文章，导出为 HTML / Excel / JSON / Markdown / TXT / Word 等多种格式。

项目完全开源，支持通过 Docker 进行 [私有部署](../advanced/private-deploy)。

## 主要特性

- **无需安装**：基于浏览器运行，打开网站即可使用
- **批量下载**：支持一次性下载整个公众号的所有文章
- **多格式导出**：HTML（含图片和样式）、Excel、JSON、Markdown、TXT、Word
- **数据丰富**：支持抓取阅读量、点赞数、评论等元数据
- **灵活过滤**：按作者、标题、发布时间、是否原创等条件筛选
- **合集支持**：支持按合集批量下载文章
- **API 接口**：提供 RESTful API，方便第三方集成

## 公共网站 vs 私有部署

公共网站和私有部署运行的是完全相同的程序，**功能完全一致**，区别如下：

|     | 公共网站 | 私有部署 |
|-----|------|------|
| 可用性 | 较高   | 高    |
| 隐私性 | 中    | 高    |
| 稳定性 | 较高   | 高    |

**可用性**：公共网站部署在 [Cloudflare](https://www.cloudflare.com) 上，国内部分地区可能存在访问限制；私有部署在本地运行，不受网络限制。

**隐私性**：公共网站后端会存储登录的公众号 Cookie 数据（用于 API 调用等功能）；私有部署通过 Docker 磁盘映射，所有数据完全保存在本地。

**稳定性**：公共网站可能因域名变更等原因导致中断；私有部署通过固定的端口映射，访问地址始终不变。

::: warning 关于数据与域名的关系
本工具的文章数据存储在浏览器的 IndexedDB 中，该数据库与访问域名绑定。如果域名变更，之前缓存的数据将无法直接访问。因此建议使用私有部署或确保访问域名的一致性。
:::
